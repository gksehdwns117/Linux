
** 기본적으로 사용하는 shell 	: /bin/bash
** alias 영구등록 파일		: ~/.bashrc
** wheel 그룹			: 관리자 권한을 대행하는 그룹

<명령어>
export		
	+ PATH(인자값)	: 경로를 지정하는 변수(미리 지정해놓은 메모리의 값)
env			: 운영체제의 변수들을 출력하는 명령어

uset			: 지정 된 변수의 값을 해제(제거)하는 명령어
type
	+ 명령어 파일명	: 해당 파일이 내장된 명령어인지, 외부명령어인지 출력하는 명령어
			> 'builtin' - 내부명령어	'aliased...' - 외부명령어
	[옵션]
		??
ls	[옵션]
	-h		: 해당 dir 내부의 정확한 파일크기
	-F		: 해당 파일의 타입(실행파일,)
	-s		: 해당 파일의 블럭사이즈를 출력
			> 블럭 : 디스크의 할당공간 , 실제 파일사이즈보다 더 큰 크기
			> 하나의 블럭의 크기는 4k
** ls 명령어는 기본적으로 dir,파일 이름 기준 오름차순으로 나타냄
	-r		: 이름 기준 내림차순으로 출력
man			: 명령어의 정보와 옵션들을 편집기 환경으로 보여주는 명령어
	+ 명령어		: 해당 명령어의 자세한 정보를 출력
wc	[옵션]		
	-c		: 전체 문자의 수 출력
	-l		: 전체 라인의 수 출력
	-w		: 전체 단어의 수 출력	
tee			: 터미널에 출력된 내용들을 파일을 생성하고 저장
stat			: 해당 파일의 모든통계를 출력
rm	-i		: 삭제 여부를 묻는 옵션
source			: 스크립트 파일을 수정 한 후 수정된 값을 적용하기 위한 명령어
file			: 파일의 형태를 출력하는 명령어
cat	-n		: cat 파일 내용 출력에 라인넘버를 부여
nl			: (= cat -n)
