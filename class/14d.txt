** 리눅스 백업파일
	> daily backup , monthly backup ... 
	> 실무 업무중 매일 하게 되는 중요한 이론
	> tar 형식의 파일로 압축
** '|' 	파이프(pipe) : (=메모리 버퍼)
	> 왼쪽 명령어의 출력문을 메모리 버퍼(임시공간)에 저장하고 뒤의 명령문의 인자값으로 보내진다
	> 최종의 결과값을 표준출력 하기 위해서 마지막 파이프라인은 생략한다
** '^'  : 위치에 따라 처음을 뜻하기도 하고 not(부정)을 뜻하기도 함
	> 하나의 문자열 집합의 바깥에 존재하면 처음 문자를 뜻함
	> 문자열 집합의 안에 존재하면 not을 뜻함
** '*'	: 전체를 뜻하기도 하지만 지시자로서 0개 이상을 의미하기도 함





<명령어>
grep word(정규표현식) 파일이름
	> 해당 파일의 전체내용을 뒤져 해당 word에 해당하는 내용을 행단위로 출력
	[옵션]
	-i			대소문자를 구분하지 않고 출력
	-n			행 번호와 함께 출력
	-v			해당 word가 존재하지 않는 행만 출력
	-w			하나의 단어로 취급하여 검색
	-c			찾아낸 결과값의 총수를 숫자로 출력	( = | wc -l )
	-E			egrep의 역할
	-F			fgrep의 역할

egrep word(정규표현식) 파일이름	: grep에서 제공하지 않는 메타문자를 사용 할 수 있음
	n+			n이 한번 이상 			 
	-
	a|b			a or b
	()			정규표현식을 묶어주는 메타문자
	\ (X)			grep에서 썼던 escape문자를 사용하지 않음

fgrep '문자열' 파일이름		:grep과 역할이 동일하지만 메타문자를 사용하지 않음 
				>메타문자를 문자로 취급하고 싶을 때 사용(특수문자 검색)
				

tee(T)	: 터미널에 있는 출력문을 인자값인 파일에 저장하는 명령어 (명령어 실행 중간)
mkfifo	: 파이프를 생성해주는 명령어 (?)
	> 프로세스간 연결 시 사용
