
** /bin/bash(Bash Shell)
	: 내장명령어 저장 파일
** 명령어 치환
** 문자열로 처리 > '' 싱클쿼트사용
   문자열과명령어 병행 > "" 더블쿼트사용
** shebang
** source / bash 비교
	> source명령어는 외부 쉘에 영향을 줌
	> bash 명령어는 외부 쉘에 영향을 주지 않음






<명령어>
dd 					블록 단위로 파일을 복사하거나 파일변환 명령어
touch -t 	n(ex.yyyymmdd.ss)	해당 날짜로 파일 생성
echo $SHELL				현재 사용중인 쉘 출력
	(=/etc/passwd 파일의 사용자정보 맨끝단어)
jobs	-l				현재 백그라운드에서 작업중인 프로세스 출력
fg	백그라운드넘버			해당 백그라운드 넘버에 해당하는 프로세스 출력
	>'ctrl + c' 해당 프로세스 종료

>예제
for문을 이용하여 디렉터리 생성
만약 기존 디렉터리가 존재한다면 삭제하고 생성
os/linux,solaris,windows,novell/files/
