
** 시간 동기화 방식
	- 기존 서버를 이용한 동기화
	- ntp서버의 데몬을 직접 운영하여 동기화
** touch 명령어는 
	- 파일을 만드는 명령어
	- 기존의 파일의 시간을 바꿔주는 명령어

** 리눅스에서 '!' 기호는 부정(not)을 뜻함
	> 옵션에서 사용시 , 바로 뒤의 옵션에만 not 적용
** 와일드카드 문자
	'*' 	: 모든 문자열 의미, 모든 자릿수
			> '*n' : n으로 끝나는 모든 문자열
			> 'm*n': m으로 시작하고 n으로 끝나는 문자열
			> 'n*' : n으로 시작하는 모든 문자열
	'?' 	: 문자열의 자릿수 (개당 하나)
			ex. 01???? : 01로 시작하며 4자리가 연속되는 문자열
	[0-9]	: 아라비아 숫자
			ex. *[0-3] : 0~3으로 끝나는 문자열(비연속적)
			    *[1,3] : 1 or 3으로 끝나는 문자열
	[a-z]	: 소문자 한 자릿수
	[A-Z]	: 대문자 한 자릿수
			-> []안의 숫자,알파벳 혼용 가능
	
** 와일드카드 문자와 일반문자를 구분 = '\' escape 문자 사용
	> "?"는 일반문자 ?는 와일드카드 문자
	> \? 는 일반문자 ?는 와일드카드 문자
	

<명령어>
rdate	[옵션]	{time.bora.net | 동기화 할 서버 ip주소}
	-p	호스트의 정보만 출력하고 설정은 안함	
	-s	설정만 하고 출력 안함
	-v	상세한 정보 출력
find 	[옵션]
	-user 	소유권		해당 소유권을 가지는 파일/dir 출력
	-empty			비어있는 파일 출력
	-newer 	비교파일/dir	해당 비교파일보다 더 나중에 생성된 파일/dir 출력
	-exec 	{} 		find에서 찾은 결과값을 {}에 저장
	-perm	허가권		해당 허가권을 가진 파일/dir 출력
	-size	+-n		n보다 크고 작은 파일/dir 출력		
	

<용어>
원자시계	: 원자의 진동수가 일정함을 이용하여 만든 시계,온도등의 영향을 받지 않아 매우 정확
ntp서버	: Network 상에 연결된 장비와 장비 간에 시간 정보를 동기화하기 위한 프로토콜
